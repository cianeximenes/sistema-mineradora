## Descrição do Projeto

O Sistema Integrado da Mineradora foi desenvolvido para centralizar e otimizar os processos operacionais de uma empresa do setor mineiro. A aplicação permite o gerenciamento de informações relacionadas a cidades, equipamentos, colaboradores e ordens de serviço, proporcionando maior controle, organização e eficiência nas operações.

A interface foi construída utilizando React e Vite, oferecendo uma experiência moderna, rápida e responsiva. O armazenamento e gerenciamento dos dados são realizados através do Supabase, utilizando um banco de dados PostgreSQL.

---

## Requisitos para Compilar e Executar

Antes de começar, certifique-se de possuir as seguintes ferramentas instaladas:

### Software Necessário

* Node.js (versão 18 ou superior)
* npm (instalado juntamente com o Node.js) ou Yarn
* Visual Studio Code (VS Code) ou outro editor de código compatível

## Instalação e Execução

### 1. Acessar a Pasta do Projeto

Abra o terminal e navegue até o diretório onde o projeto está localizado:

```bash
cd frontend
```

### 2. Instalar as Dependências

Execute o comando abaixo para instalar todas as bibliotecas necessárias definidas no arquivo `package.json`:

```bash
npm install
```

### 3. Executar em Ambiente de Desenvolvimento

Para iniciar o servidor local utilizando o Vite:

```bash
npm run dev
```

Após a execução, o terminal exibirá um endereço semelhante ao seguinte:

```text
http://localhost:5173/
```

Abra esse endereço em seu navegador para acessar a aplicação.

## Estrutura Geral do Sistema

O sistema é composto pelos seguintes módulos principais:

* Gestão de cidades
* Gestão de equipamentos
* Gestão de colaboradores
* Gestão de ordens de serviço
* Dashboard de monitorização
* Relatórios operacionais
* Integração com banco de dados Supabase
